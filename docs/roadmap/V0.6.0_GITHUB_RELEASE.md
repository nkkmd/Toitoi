# Toitoi v0.6.0 — Offline SPA/PWA Golden Path

v0.6.0 integrates Toitoi's existing Canonical Event, AI annotation, Inquiry Draft, human review, provenance, lineage, storage, and multi-transport boundaries into an installable mobile-first SPA/PWA.

## Highlights

- capture field observations on a smartphone while offline
- retain local observations in IndexedDB across reloads
- use an explicit synchronization queue with visible retry state
- confirm location, personal information, and private context before synchronization
- review AI-generated inquiry candidates through accept, edit, and reject operations
- promote accepted candidates into Inquiry Drafts without confusing annotation acceptance with publication approval
- submit, approve, reject, and publish Inquiry Drafts through Standard API mutations
- enforce the existing publication guard so only independently approved Drafts can become Canonical Events
- persist published Canonical Events through the existing append-only storage modules
- deliver publications through the existing multi-transport fan-out runtime
- retain delivered, skipped, and quarantined transport results in publication provenance
- install and operate the reference frontend as a PWA

## Golden Path

```text
field observation
  → offline IndexedDB save
  → explicit synchronization
  → asynchronous AI inquiry candidates
  → human annotation review
  → Inquiry Draft promotion
  → independent publication review
  → approved Canonical Event
  → append-only storage
  → multi-transport delivery
  → provenance and lineage inspection
```

## Validation

- complete workspace CI passed on the release candidate branch
- v0.3.0-v0.5.0 Golden Path regressions remain covered
- v0.6.0 offline observation, network failure, and retry flow is deterministic
- workflow HTTP mutation and publication guard are tested
- canonical storage and multi-transport publication adapter is tested without external services

## Runtime profile

```bash
TOITOI_STORAGE_DIR=/path/to/storage \
TOITOI_AI_STORAGE_DIR=/path/to/ai-storage \
TOITOI_PROTOCOL=nostr \
corepack pnpm --filter @toitoi/api start
```

Live transport credentials are optional. Missing or failed transports are reported as skipped or quarantined rather than bypassing the human publication boundary.

## Known limitations

- production authentication, authorization, and rate limiting are not included
- workflow Draft state is a reference single-process runtime; multi-user coordination is not provided
- live external transport availability is outside deterministic CI
- semantic search, embeddings, vector search, RAG, and graph inference are not included
- AI output is not treated as agriculturally verified advice
- unattended AI publication remains unsupported
