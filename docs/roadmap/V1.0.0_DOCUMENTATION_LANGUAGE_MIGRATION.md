## Objective

Bring the maintained Toitoi documentation into a deliberate language structure before v1.0.0:

- major public and operational documents are bilingual, with English followed by equivalent Japanese;
- detailed technical and implementation documents are English;
- historical records retain their existing language unless they create a current misunderstanding.

This migration does not require translating every existing file. It requires making the maintained document set intentional, navigable, and safe for release.

## Non-blocking Class B normalization

The following should be English, but individual remaining Japanese passages do not block v1.0.0 unless they conflict with a Class A contract or operation:

- transport-specific specifications;
- package-internal READMEs;
- adapter and converter implementation documents;
- fixture and test explanations;
- release plans, implementation records, and final-review records;
- CI and deployment internals;
- detailed API endpoint reference;
- pilot internal collection and analysis templates.

For these documents:

- new text should be written in English;
- mixed-language files should be normalized when materially edited;
- broad mechanical translation without technical review should be avoided.

## Class C historical documents

Historical documents retain their existing language and structure.

A Class C file is changed only when:

- it is incorrectly presented as current;
- it points to a removed or unsafe procedure;
- it conflicts with a current normative contract;
- its language causes a material release or operational misunderstanding.

## Release decision rule

The language migration gate passes when:

- all registered Class A documents exist;
- all release-blocking Class A documents contain English followed by equivalent Japanese;
- critical operational literals match;
- default CI passes on the exact final documentation head;
- a human reviewer confirms no known material divergence.

Class B or Class C translation completeness is not a v1.0.0 release condition.
