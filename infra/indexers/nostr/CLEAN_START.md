# Nostr Indexer Clean Start Wrapper

`infra/indexers/nostr/` のクリーンスタート補助資料です。

共通のクリーンスタート手順は [../CLEAN_START.md](../CLEAN_START.md) を参照してください。

## 使いどころ

- Nostr 運用から移行する前後の整理
- transport ingest と indexer の停止順を確認したいとき
- Nostr 固有の補助資料を読みたいとき
